## Title of the Project

## Haven – A Safe Space for Mental Health
A platform combining stress prediction and empathetic AI chat support for mental well-being.

## About

Haven collects users’ biometric and lifestyle data (sleep, heart rate, steps) to predict stress levels using a Random Forest model. Based on the stress prediction, an AI chatbot powered by Llama 3.1 via Groq provides empathetic, context-aware support in real-time, promoting mental health awareness and self-care.

## Features

Real-time stress prediction (Low, Medium, High).

AI chatbot for personalized support.

Biometric and lifestyle tracking.

Mobile-friendly responsive interface.

Instant responses via Groq-powered LLM.

## Requirements

OS: Windows 10

Languages/Frameworks: Python 3.8+, Flask , GROQAPI , HTML , CSS , JS 

ML Libraries: scikit-learn, TensorFlow , PyTorch

IDE & Tools: VSCode, Git and Github

## System Architecture


Flow: User inputs → Stress Prediction → Contextual AI Response → Display.

## Output
Stress Prediction

Chatbot Response

Prediction Accuracy: 93–95%
Response Latency: <500ms

## Algorithms

Random Forest: Classifies stress level from lifestyle & biometric data.

Llama 3.1: Generates empathetic, context-aware chatbot responses.

## Impact

Provides instant mental health support, fosters self-care, and enables a safe, inclusive environment for users.
